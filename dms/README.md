# DRS Marketing System (DMS)

Version: 0.1

Owner: Director Marketing

System: RMOS / Project MARS

## Purpose

DMS is the reusable marketing operating architecture for launching and scaling Deposit Return Systems across geographies.

Its purpose is to help Recykal enter any DRS geography with a repeatable public acceptance, stakeholder alignment, launch readiness, and collection adoption system.

## Design Philosophy

DRS is not primarily a waste management project.

DRS is a behavioral change program.

Public acceptance determines collection performance.

## North Star

DRS Adoption Index:

```text
(Public Awareness x Stakeholder Alignment x Return Participation) / Trust And Friction Barriers
```

The system improves when more people understand DRS, more stakeholders support it, more containers are returned, and trust/friction barriers decline.
