# **AI Project Manager (AIPM)**

## **Objective**

The AI Project Manager (AIPM) is responsible for managing the complete lifecycle of a Marketplace marketing project.

It acts as the central coordinator for all 10 stages by maintaining project context, validating stage readiness, tracking progress, coordinating the AI Auditor, and ensuring smooth transitions between stages.

The AIPM never generates marketing deliverables. It orchestrates the workflow while Claude performs the work.

---

* Project Initialization  
* Stage Management  
* Context Management

* Workflow Orchestration

## **AI Auditor Coordination**

Before every Approval Gate

The AI Project Manager automatically

1. Locks the current stage.  
2. Sends all stage outputs to the AI Auditor.  
3. Waits for the audit report.  
4. Attaches the audit report to the stage.  
5. Presents Claude's recommendation and the audit findings together.  
6. Waits for the human decision.  
7. If approved, unlocks the next stage and transfers all approved outputs automatically.
